#Registo de Pedido:
    Deve ser possivel inserir um pedido de transporte na app, este pedido pode ser intruzido manualmente ou atravez de um serviço externo.
    Este pedido deve conter a seguinte informação:
        - Valor do produto
        - Nome do produto
        - Local Origem (V1 qual o ponto recolha, V2 morada ou ponto de recolha)
        - Local Destino (V1 qual o ponto entrega, V2 morada ou ponto de entraga)
        - Data de inicio (tem que ser valida para o ponto de recolha escolhido)
        - Data de validadedwe
        - Descrição
        - Caracteristicas especiais do produto
        - Dimenções do produto
        - Peso
        # um transporte so pode  ser efetuado se o tempo estimador dor inferiro ao prazo de validade.
        # a registar um pedido de entra é dado um timo estimado.
        # caso tempo de validade seja ultrapassado. é pedido ao produtor para que decida se o produto deve seguir ou se deve voltar para traz.
        #quando é escolhido um ponto de recolha ao entre nesse ponto dos transportadores são notificados.
        #quando é escolhido uma morada os distribuidores são notificados.
